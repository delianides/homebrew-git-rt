# homebrew-git-rt

Homebrew tap for [git-rt](https://github.com/delianides/git-rt) — a real-time terminal dashboard for git changes.

## Install

```bash
brew tap delianides/git-rt
brew install git-rt
```

## Update

```bash
brew update
brew upgrade git-rt
```

## Uninstall

```bash
brew uninstall git-rt
brew untap delianides/git-rt
```
